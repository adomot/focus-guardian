## 作品タイトル

Focus Guardian — サボりを見逃さない、あなた専属の集中監視AIエージェント

## 概要（〜200字目安）

資格勉強などの作業中、気づけばスマホ…を終わらせる監視AIエージェント。あなたが自由な言葉で伝えた「悪習慣」を Gemini が画像判定できる条件に変換し、Webカメラの映像を60秒ごとに自律判定。サボりを検知すると、事前に選んだ BGM や声かけを Alexa から流して作業に引き戻します。画像は一切保存しないプライバシー設計。

## ストーリー

### ① 解決したい課題と背景

「今日こそ勉強するぞ」と机に向かったのに、気づけばスマホを30分——意志力に頼った集中は、あまりにも簡単に崩れます。ポモドーロタイマーやスクリーンタイム制限などの既存ツールは「自己申告」や「特定アプリのブロック」止まりで、**机の上の現実の行動**（スマホいじり、居眠り、離席）までは見てくれません。

人間の「見られていると頑張れる」性質（ホーソン効果）を、AI エージェントで再現できないか。それが Focus Guardian の出発点です。

### ② 想定ユーザー

- 資格試験・受験勉強など、明確な目標があるのに一人だと集中が続かない人
- 在宅勤務・在宅学習で「見てくれる人」がいない環境の人
- スマートスピーカーを持っていて、生活の中に自然に介入してほしい人

### ③ プロダクトの特徴

**1. 監視対象はあなたの言葉で決まる**
「スマホをいじってしまう」「居眠りしてしまう」——自由入力の悪習慣を ADK 上の StructuringAgent が「手にスマートフォンを持ち視線が手元にある」のような**画像判定可能な視覚条件に構造化**します。ユーザごとに監視対象が違うため、固定の分類モデルでは作れない、生成 AI エージェントならではの仕組みです。

**2. 知覚 → 判断 → 行動を自律で回し続ける**
ブラウザが60秒ごとに Web カメラ画像を送信し、JudgeAgent（Gemini 2.5 Flash Lite のマルチモーダル理解）が「集中 / 悪習慣 / 不在」を判定。連続検知・確信度・不在の区別・再介入の抑制まで**エージェントが自分で介入タイミングを決め**、介入後は「作業に戻れたか」まで評価・記録します。

**3. 介入はあなたが選んだ方法で、Alexa から**
ヒアリングで選んだ BGM、またはあなたが決めたフレーズ（Cloud Text-to-Speech で音声化）を、Voice Monkey 経由で Amazon Echo から再生。スピーカーがなくてもブラウザ再生にフォールバックします。

**4. プライバシーへの配慮**
カメラ画像は判定のリクエスト内でのみ使われ、**どこにも保存されません**。Firestore に残るのは構造化された判定結果と介入履歴だけ。Gemini へは学習に使われない Vertex AI 経由で送信します。

**5. DevOps: つくる・まわす・とどける**
GitHub 公開リポジトリ + GitHub Actions で、テスト（バックエンド24件+lint）→ Cloud Run デプロイ → ヘルスチェックまで main へのマージで自動実行。カメラ（Web カメラ → Nest Cam）もスピーカー（Alexa → Google Home）もアダプタ設計で差し替え可能な、運用と拡張を見据えた構成です。
